device-tool
========
Simple tool for detecting device info and bots.


## Setup

Install
```
npm install device-tool
```

## Usage
Import package into your project
```
import device from 'device-tool'
```

Example:
- device.macos()
- device.ipod()
- device.ipad()
- device.iphone()
- device.IsIos()
- device.android()
- device.androidPhone()
- device.androidTablet()
- device.IsAndroid()
- device.blackberry()
- device.blackberryPhone()
- device.blackberryTablet()
- device.windows()
- device.windowsPhone()
- device.windowsTablet()
- device.mobile()
- device.tablet()
- device.desktop()
- device.portrait()
- device.landscape()
- device.IsOrientation()
